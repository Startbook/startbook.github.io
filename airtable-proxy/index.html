Airtable proxy

Server-side proxy that whitelists fields from Airtable and exposes them to the Startbook judging app. Requests are CORS-restricted to approved Startbook origins.

Endpoints

GET /api/applicants

Applicants for a given judging year, filtered to the Presentation view.

Query: year (4-digit, defaults to 2026)

GET /api/attendees

Judges & staff attending the judging day, plus organisation and attendee-type choices from the Airtable schema.

GET /api/attachment

Streams an Airtable attachment with Content-Disposition: inline so PDFs render in an iframe instead of downloading.

Query: url (required, must be an airtableusercontent.com or dl.airtable.com URL)

Internal service. This proxy is for use by the Startbook judging app only. Calls from other origins will be blocked by CORS.
